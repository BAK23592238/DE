### Project Summary
The project implements a complete data engineering workflow:

1. Web scraping book data (title, author, year, rating, price) using Python.
2. Structuring and storing data in a MySQL relational database.
3. Automating data movement and transformation using Apache NiFi.
4. Converting data into JSON format and integrating with MongoDB.
5. Evaluating query execution performance and comparing SQL vs NoSQL approaches.

The system demonstrates structured storage, automation, distributed data handling concepts, and performance evaluation aligned with the module learning outcomes.

### Core Features
= Automated Python web scraper (Requests + BeautifulSoup + Pandas)
- CSV data generation (techreads_books.csv)
- MySQL database creation and structured schema design
- SQL querying with sorting and indexing evaluation
- Apache NiFi automated dataflow pipeline
- JSON data conversion and MongoDB integration
- SQL vs NoSQL query performance comparison
- Schema diagram (draw.io)
- Documented analysis and reflection
